Featured Projects

Selected projects that demonstrate offensive testing, technical depth, and measurable outcomes.

Attack Surface Mapper for Modern Web Apps

A reconnaissance project built to enumerate subdomains, fingerprint technologies, collect endpoints, and prioritize exposed attack surface across multi-service web environments.

Tech Stack Python, Bash, httpx, Subfinder, Katana, JSON reporting
Result Reduced manual recon time and produced structured target inventories for faster follow-up testing.

Broken Access Control Lab Simulation

A controlled web application lab designed to reproduce IDOR, horizontal privilege escalation, and role-based access control failures for assessment practice and demonstration.

Tech Stack Node.js, Express, JWT, Docker, Burp Suite
Result Built repeatable exploit scenarios and documented test cases aligned with real pentest methodology.

Automated XSS Payload Validation Framework

A lightweight testing framework for validating reflected and stored XSS behavior across parameters, rendering contexts, and common input filtering patterns.

Tech Stack Python, Playwright, HTML parsing, custom payload sets
Result Improved coverage during input testing and quickly highlighted exploitable rendering paths.
